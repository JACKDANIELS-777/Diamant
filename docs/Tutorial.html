üîπ Overview

Diamant is a custom programming language designed to combine the simplicity of Python with the power and control of systems languages like C. It features:

  • Optional strong typing
  • Memory tracking for variables
  • Python interoperability
  • Function overloading
  • Custom infix operators
  • Control over execution with blocks like TIME, WATCH, DEFER, and ONCE

‚úÖ Type Declarations & Casting

INT a = 10;
FLT b = (FLT) a;
BOOL c = (BOOL) 5;

Strong typing can be enforced with keywords like INT, FLT, STRING. Casts are done using C-style syntax.

üì¶ Memory Tracking (MEM)

MEM x = 5;
x = 10;
x = 20;
print(x@0); // 20
print(x@1); // 10
print(x@2); // 5

MEM enables history tracking of a variable. You can query the past values with x@index.

üîÅ Looping Features

for i in [1..5] {
    print(i)
};

while x < 10 {
    x += 1
};

for i,v in [1,2,3] {
    print(i,v)
};

üïí TIME Block

TIME t {
    for i in [1..100000] {
        a = i
    }
};
print(t);

Measures how long a block of code takes to run.

üßµ DEFER Block

fun myFunc() -> {
    DEFER { print("Done!") },
    print("Running...")
};
myFunc();

DEFER ensures cleanup or final steps after a function completes.

üïπÔ∏è ONCE Block

fun init() -> {
    ONCE { print("This only runs once") }

üëÄ WATCH Block

MEM a = 0;
WATCH a {
    print("a changed to", a)
};
a = 1;
a = 2;

Watch a variable and execute logic when it changes.

üß† Function Overloading

fun greet() -> { print("Hi") };

greet();

üîß Custom Infix Operators

INFIX "sum"(a, b) => {
    ret a + b + 10
};

a = 1; b = 2;
print(a "sum" b); // prints 13

Powerful user-defined infix operators using strings like "sum".

üì¶ Python Integration
