Prototype Interpreter

Welcome to the prototype interpreter for Diamant, a new programming language that’s still in its early development stages. Our goal is to create a powerful and user-friendly language for developers that combines simplicity with flexibility. Will be available in Afrikaans.

Current Status

As of 2025/03/18. Diamant is still in its super early development stages. The interpreter is functional but only supports a few basic features. There’s a lot more to come!

What's Next?

In the near future, we’ll be adding more features and improving the interpreter to handle more complex programs. Stay tuned for updates!
