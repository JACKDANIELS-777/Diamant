Overloading in Diamant

  • Function Overloading — Define multiple functions with the same name but different parameters.
  • Operator Overloading — Use INFIX to define custom operators like "add" or "combine".

Example: Function Overloading

fun greet() -> { print("Hello!") };
fun greet(name) -> { print("Hello, ", name) };
greet();
greet("Alice");

Example: Infix Operator Overloading

INFIX "sum"(a, b) => { ret a + b };
print(10 "sum" 5); # prints 15
